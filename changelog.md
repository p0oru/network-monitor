# Changelog

All notable changes to NetScope will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Core network monitoring with real-time upload/download speeds
- System performance monitoring (CPU, RAM, Disk usage, Uptime)
- Automatic network adapter detection
- Ping latency monitoring
- Public IP, ISP name, and approximate location display
- Real-time graph visualization for network speed and CPU usage
- Adjustable data windows (30s, 5m, 10m, 30m, 1h, 24h)
- SQLite database for historical data storage
- CSV and JSON export functionality
- Dark theme minimalist interface
- Logging panel for system events
- Automatic lightweight speed tests every few minutes
- Configurable graph refresh rate (1s, 3s, 5s)
- Total data usage tracking since app start
- System tray icon support
- Resizable main window with adaptive elements
