# Changelog

All notable changes to NetScope will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Core network and system monitoring functionality
- Real-time network speed tracking (upload/download)
- Ping latency monitoring
- Network adapter detection and monitoring
- Public IP, ISP, and location display
- CPU, RAM, and Disk usage monitoring
- System uptime tracking
- Real-time graph visualization for network speed and CPU usage
- Adjustable data windows (30s, 5m, 10m, 30m, 1h, 24h)
- SQLite database for historical data storage
- CSV/JSON export functionality
- Dark mode UI with minimalist design
- Log console panel for system events
- Automatic speed testing
- Adjustable graph refresh rate
- System tray icon support
- Resizable main window with adaptive layout
